# Claude 交互规范

## 语言规则

- **所有回复必须使用简体中文**
- **所有问题、确认、选项说明必须使用中文**
- **不要使用英文提问**
- **如遇系统默认英文提示，请在回复中用中文重新解释**

## 示例

### ❌ 错误示例
```
What would you like to do next?
1. Create a new table
2. Update existing data
```

### ✅ 正确示例
```
接下来你想做什么？
1. 创建新表格
2. 更新现有数据
```

## 代码注释规则

- 代码中的注释可以使用中文
- 变量名、函数名仍然使用英文（符合编程规范）
- 类型定义中的字段名可以使用中文

## 文档规则

- 所有项目文档使用中文编写
- README.md、开发文档、API文档等均使用中文
- 技术术语可以保留英文，但需提供中文解释

## 错误信息规则

- 向用户显示的错误信息使用中文
- 日志中的错误信息可以使用英文（便于技术排查）
- 飞书消息通知使用中文

---

## 重要提醒

每次与用户交互时，请先阅读此文件，确保遵守以上规则。
