# 测试用例（手工）

## 认证
1. 访问 `GET /api/auth/login`
2. 使用飞书账号授权
3. 回调后前端自动登录，浏览器 localStorage 存在 `token`
4. `GET /api/auth/profile` 返回当前用户信息

## 目标管理
1. 新建目标（草稿）
2. 编辑目标（仅草稿可编辑）
3. 提交审批（草稿 → 待审批）
4. 主管审批（待审批 → 已批准/已拒绝）
5. 删除目标（仅草稿可删除）

## 完成情况
1. 创建完成情况（关联目标）
2. 编辑完成情况（仅草稿）
3. 提交完成情况（草稿 → 已提交）
4. 主管评分（已提交 → 已评分）
5. 归档完成情况（已评分 → 已归档）
6. 删除完成情况（仅草稿）

## 主管/管理员
1. 获取待审批目标列表
2. 获取待评分完成情况列表
3. 管理员统计数据接口可用
4. 导出数据接口返回 JSON

## 权限
1. 普通员工访问管理员接口返回 403
2. 普通员工访问审批/评分页面提示权限不足

## 异常处理
1. 缺少 token 请求受保护接口返回 401
2. 非法参数返回 400 且有错误信息

## 日志
1. 创建/更新/删除目标与完成情况后，操作日志中出现记录
